Demo CRUD API & Web Frontend Plan

Notes

  • Demo CRUD API with a simple data model: [id, name, date]
  • Date format is dd-mm-yyyy
  • Data is stored in memory with optional file-based storage
  • Applications are deployed on an OpenShift cluster
  • Node.js with Express is used for both API and frontend
  • The API is available at: https://crud-demo-api-git-erikfirsttest.apps.dekradk.dekra.nu
